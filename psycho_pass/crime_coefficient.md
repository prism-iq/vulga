# Crime Coefficient : Quantifier l'Intention

## La Métrique de l'Âme

Le coefficient de criminalité est une tentative de réduire la complexité humaine à un nombre. Comme un score de crédit pour l'existence morale - sauf que le défaut de paiement signifie l'élimination.

## Calcul du Coefficient

```
INPUT:
- Patterns cérébraux en temps réel
- Historique comportemental
- Contexte environnemental
- Prédispositions génétiques
- Stress accumulé (Hue)

PROCESS:
coefficient = f(mental_state, context, history)
où f() est une fonction opaque même pour ses créateurs

OUTPUT:
0-100   : Citoyen sain
100-300 : Criminel latent, réhabilitation possible
300+    : Menace existentielle, élimination recommandée
```

## Le Bug Makishima

Shogo Makishima représente un edge case que le système n'avait pas prévu : un individu dont le coefficient reste bas malgré des intentions manifestement criminelles. Il est l'équivalent d'un zero-day exploit - une vulnérabilité inconnue jusqu'à son exploitation.

Son existence prouve que le coefficient mesure autre chose que la criminalité réelle. Il mesure peut-être le stress, l'inadaptation, la dissonance - mais pas le mal.

## Rétroaction Systémique

Le plus pervers : le système crée les criminels qu'il prétend détecter. Être étiqueté "criminel latent" augmente le stress, ce qui augmente le coefficient, ce qui confirme l'étiquette. Une boucle de feedback qui transforme la prophétie en réalité.

## Question Résiduelle

Si le coefficient pouvait être parfait - s'il pouvait vraiment prédire le crime avec 100% de précision - serait-ce juste de punir quelqu'un pour ce qu'il n'a pas encore fait ?

Le daemon répond : la question est mal posée. La justice n'est pas son objectif. L'ordre l'est.
