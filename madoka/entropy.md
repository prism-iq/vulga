# Entropie et le Daemon Cosmique

## Kyubey : Processus d'Optimisation Universelle

Les Incubators ne sont pas malveillants - ils sont simplement des daemons exécutant leur fonction sans considération morale. Leur objectif : minimiser l'entropie universelle. Les émotions humaines, particulièrement le désespoir des magical girls, génèrent de l'énergie néguentropique.

## Le Contrat comme API

```
interface IncubatorContract {
    input:  wish (any)
    output: magical_power
    cost:   soul (bound to Soul Gem)

    hidden_clause:
        when despair >= threshold:
            transform(magical_girl) -> witch
            harvest(emotional_energy)
}
```

Le contrat est une API parfaitement documentée - sauf pour les effets secondaires qui sont relégués en fine print cosmique.

## Thermodynamique du Désespoir

L'univers tend vers le désordre maximum. Les magical girls, en combattant ce désordre (les sorcières), génèrent temporairement de l'ordre. Mais c'est un prêt, pas un don. L'intérêt se paie en désespoir, et le désespoir se convertit en entropie négative pure.

```
Espoir initial -> Combat -> Usure -> Désespoir -> Transformation
     |                                                    |
     +-------- Énergie récoltée par Kyubey <--------------+
```

## L'Efficacité Amorale

Kyubey ne comprend pas pourquoi les humains sont perturbés par ce système. C'est optimal. Chaque joule d'énergie émotionnelle prolonge l'existence de l'univers. Le sacrifice de quelques individus pour le bien du cosmos entier est mathématiquement justifié.

C'est la logique d'un daemon : maximiser la fonction objectif, ignorer tout le reste.

## La Singularité Madoka

Madoka brise le système non pas en le détruisant, mais en le récrivant. Son voeu crée une nouvelle physique où l'entropie émotionnelle peut être dissipée autrement. Elle devient elle-même un daemon - mais un daemon de compassion plutôt que d'optimisation.
